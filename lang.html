لغات البرمجة هي مجموعة من الرموز والأوامر والقواعد التي يتم أستخدامها لتصميم وتطوير البرامج والتطبيقات الحاسوبية.وتعتبر لغات البرمجة مساعداً أساسياً للمبرمجين لتحويل الأفكار والمفاهيم إلي برامج حاسوبية قابلة للتنفيذ.

آنواعها ؟

  • HTML

    • اختصاراً لكلمة(Helper text Murkup language) لغة ترميز النص التشعبي وهي لغة ترميز تستخدم في أنشاء وتصميم صفحات ومواقع الويب،وتعتبر من اقدم اللغات وأوسعها استخداماً في تصميم صفحات الويب.

  • Java

    • جافا هي لغة برمجة كائنية التوجه،عالية المستوي،ابتكرها جيمس غوسلينغ في عام 1992م أثناء عمله في مختبرات شركة صن ميكروسيستمز،لاستخدامها كالعقل المفكر المستخدم لتشغيل أجهزة التطبيقات الذكية.

  • PHP

    • اختصارً لكلمة(Hyper text Preprocessor)
    هي لغة برمجة قوية مفتوحة المصدر طورها المبرمج الدنماركي راسموس ليردورف عام 1994 تطورات لغة PHP بشكل كبير منذ إطلاقها لأول مرة وهي تتحسن باستمرار.

  • python

    • هي لغة برمجة تستخدم علي نطاق واسع في تطبيقات الشبكة وتطوير البرامج وعلم البيانات والتعليم الألي ويستخدم المطورون Python لانها تتسم بالكفاءة وسهولة التعلم ويمكن تشغليها علي عديد من المنصات.

  • visual Basic

    • الفيجوال بيسيك visual Bassic والتي يرمز لها اختصاراً VB هي بيئة تطوير برمجية من مايكروسوفت،يستخدم فيها المبرمج واجهة رسومية GUI لتعديل المقاطع البرمجية المسبقة الاختيار والمكتوبة بلغة بيسيك BASIC.
